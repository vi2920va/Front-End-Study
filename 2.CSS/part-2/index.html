CSS 케스케이드(Cascade)

케스케이딩(Cascading) 이란?

cascading

The process of combining several style sheets and resolving conflicts between them.

-Håkon Wium Lie, Cascading Style Sheets, PhD Thesis

  • class selector
    attribute selector
    pseudo-class
    Specificity: 0,1,0 (10)

  • id selector
    Specificity: 1,0,0 (100)

    Specificity: 1,0,0,0 (1000)
