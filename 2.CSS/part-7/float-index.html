CSS 레이아웃(Layout)

뷰포트(Viewport)와 관련하여 박스를 레이아웃하는 방법을 학습해봅니다. 대표적인 레이아웃 테크닉인 플로팅(Floating)과 포지셔닝(Positioning)을 먼저 학습하고, 플렉스박스(Flexbox)와 그리드(Grid) 레이아웃에 대해 공부해봅니다.
